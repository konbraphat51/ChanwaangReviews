# Spatial Presence and Emotions during Video Game Playing: Does it Matter with Whom You Play?

* Earned insights on how the nature of the opponent (computer, friend, stranger) influences Spatial Presence, emotional responses, and physiological arousal by using self-report ratings and cardiac interbeat intervals (IBIs). 


* 自己報告評価と心拍間隔（IBI）の測定により、対戦相手の性質（コンピュータ、友人、見知らぬ人）が空間的プレゼンス、感情反応、および生理的覚醒にどのように影響するかという知見を得た論文

## 概要

本研究は、ビデオゲームをプレイする際の対戦相手（コンピュータ、友人、見知らぬ人）の違いが、プレイヤーの「空間的プレゼンス（Spatial Presence）」、感情反応、および脅威と挑戦の評価に与える影響を検証しました 。参加者は被験者内計画（within-subjects design）において、2種類の異なるゲームをそれぞれの対戦相手とプレイしました 。評価には自己報告式のアンケートに加え、生理的覚醒の指標として心拍間隔（IBI）が用いられました 。その結果、コンピュータとの対戦に比べ、人間との対戦は、より高い空間的プレゼンス、エンゲージメント、予想される脅威、プレイ後の挑戦の評価、および生理的覚醒を引き起こし、感情的な反応もよりポジティブであることが示されました 。さらに、見知らぬ人との対戦に比べ、友人との対戦は、より高い空間的プレゼンス、エンゲージメント、および自己報告と生理指標の両方における覚醒を引き起こしました 。これらの結果は、対戦相手の性質が、おそらく覚醒や注意プロセスへの影響を介して、空間的プレゼンスに影響を与えることを示唆しています 。

## 背景

### 研究の意義

* メディアユーザーが、表示された環境の中に個人的かつ物理的に存在していると感じる体験は「プレゼンス（Presence）」、特に「空間的プレゼンス（Spatial Presence）」と呼ばれます 。


* これまでの研究では、プレゼンスの決定要因として、メディアの形式的特徴（感覚刺激の数、画質、画面サイズなど）やコンテンツの特徴（社会的リアリズムなど）が特定されてきました 。


* 本研究は、対戦相手という「社会的文脈」が、ビデオゲームプレイ中のプレゼンス体験や感情反応にどのような違いをもたらすかを明らかにする点に意義があります 。具体的には、「コンピュータ、友人、見知らぬ人の誰とプレイするかによって違いが生じるか？」という問いに取り組みました 。



### 関連研究

* 
**プレゼンスと感情:** 空間的プレゼンスは感情の「価（valence）」と「覚醒（arousal）」の両次元に影響を与える可能性があります 。高いプレゼンスを生むメディア表現は、しばしば高い自己報告的および生理的覚醒を引き起こすことが示唆されています 。


* 
**社会的相互作用と覚醒:** 社会的相互作用は関与や熱意により覚醒を引き起こす可能性があります 。他者との競争的なゲームは、マシン相手よりも高い心拍反応を引き起こすことがわかっています 。


* 
**社会的促進と評価:** 他者（友人や見知らぬ人）の存在下でタスクを行う場合、観察者がいない場合と比較して交感神経系の覚醒が高まることが予想されます 。これは、観察する他者の存在が、タスク遂行に対する「評価の可能性（evaluation potential）」を生み出すためです 。


* 
**友人 vs 他人:** 友人からのソーシャルサポートは心血管反応を抑制するという研究がある一方で 、競争的なビデオゲームにおいては、友人が継続的なタスク遂行のリマインダーとして機能し、見知らぬ人よりも高い覚醒をもたらす可能性があります 。


* 
**注意資源とプレゼンス:** 空間的プレゼンスの2レベルモデルによれば、媒介された環境への注意資源の集中的な配分が空間的プレゼンスの形成に寄与します 。覚醒は注意を高めるため、社会的対戦による覚醒の増加がプレゼンスを高める可能性があります 。



## 手法

### 概要

実験室環境において、参加者が3種類の対戦相手（コンピュータ、友人、見知らぬ人）と2種類のビデオゲームをプレイする実験を行いました。主観評価と生理指標（心電図）を同時に測定しました。

### 前提条件

* 参加者はフィンランドの学部生99名（男性51名、女性48名、平均年齢23.8歳） 。


* 実験は同性の3人グループで行われ、その内訳は「互いに友人である2名」と「他の2名とは面識のない1名（見知らぬ人）」で構成されました 。


* データ分析対象は、各グループから選ばれた33名の「メイン参加者」のみです 。



### 詳細

* 
**実験デザイン:** 2（ゲーム）× 3（対戦相手）の被験者内（within-subjects）デザイン 。


* **使用ゲーム:**
* 
*Super Monkey Ball Jr.* (Sega): 非暴力的で明るい雰囲気のゲーム。反射神経と戦略が必要 。    * *Duke Nukem Advance* (Take 2 Interactive): 一人称視点のシューティング（FPS）。暴力的要素を含む 。    * プラットフォームは任天堂ゲームボーイアドバンスを使用し、対戦時は通信ケーブルで接続しました 。




* **手順:**
1. 事前の質問紙への回答と同意書の記入 。


2. メイン参加者に電極を装着 。


3. 全参加者がシングルプレイヤーモードで各ゲームを5分間練習 。


4. 7分間の休憩（ベースライン生理測定） 。


5. メイン参加者が、コンピュータ、友人、見知らぬ人とそれぞれ8分間、2つのゲームをプレイ（計6セッション、順序はランダム化） 。対戦相手は隣に座ってプレイしました 。


6. 各ゲームセッションの直前・直後に質問紙に回答 。





### 評価方法

* 
**空間的プレゼンスとエンゲージメント:** ITC-Sense of Presence Inventory (ITC-SOPI)を使用 。


* 空間的プレゼンス（Spatial Presence）：19項目（例：「ゲームの世界にいるような感覚があった」） 。


* エンゲージメント（Engagement）：13項目（例：「夢中になった」） 。


* 生態学的妥当性/自然さ（Ecological Validity/Naturalness）：5項目 。




* 
**感情（価と覚醒）:** 9段階の絵画的尺度（Self-Assessment Manikinに類似）を使用 。


* 価（Valence）：しかめっ面（ネガティブ）から笑顔（ポジティブ）まで 。


* 覚醒（Arousal）：穏やかな状態から興奮した状態まで 。




* **脅威と挑戦の評価:**
* 予期される脅威（Anticipated threat）：ゲーム前に「今からのゲームはどのくらい脅威的か？」を質問 。


* 挑戦（Challenge）：ゲーム後に「プレイしたゲームはどのくらい挑戦的（やりがいがある/難しい）だったか？」を質問 。




* 
**生理指標:** 心電図（ECG）から心拍間隔（IBI: Interbeat Interval）を測定 。


* IBIが短いほど心拍数（HR）が高いことを意味し、覚醒の指標となります 。


* データは500Hzでサンプリングされ、ゲーム中の30秒ごとのエポック（区間）平均値を算出しました 。





## 結果と結論

### 概要

人間（友人または見知らぬ人）との対戦は、コンピュータとの対戦と比較して、より高い生理的覚醒、ポジティブな感情、空間的プレゼンス、およびエンゲージメントをもたらしました。また、友人との対戦は、見知らぬ人との対戦よりも高い覚醒とプレゼンスを引き起こしました。

### 詳細

* **脅威と挑戦:**
* 友人や見知らぬ人との対戦は、コンピュータとの対戦に比べ、予期される脅威が有意に高かった () 。


* 挑戦の評価も人間相手の方が高い傾向にあったが、統計的に有意な差にはわずかに届かなかった () 。




* **プレゼンス評価:**
* 人間（友人・見知らぬ人）との対戦は、コンピュータとの対戦に比べ、有意に高い空間的プレゼンスを示した () 。


* 友人との対戦は、見知らぬ人との対戦に比べ、有意に高い空間的プレゼンスを示した () 。




* **エンゲージメント:**
* 人間との対戦は、コンピュータとの対戦より有意に高いエンゲージメントを示した () 。


* 友人との対戦は、見知らぬ人との対戦より有意に高いエンゲージメントを示した () 。




* **感情（価と覚醒）:**
* 人間との対戦は、コンピュータとの対戦より有意にポジティブな（価が高い）感情反応を引き出した () 。友人対見知らぬ人の差は有意傾向のみ () 。


* 自己報告による覚醒度は、人間対コンピュータの全体比較では有意差がなかった () が、これは見知らぬ人との対戦時の覚醒度が低かったためである 。


* 友人との対戦は、見知らぬ人との対戦よりも有意に高い自己報告覚醒度を示した () 。




* **生理指標（IBI）:**
* 人間との対戦は、コンピュータとの対戦に比べ、IBIが有意に短かった（＝心拍数が高かった、生理的覚醒が高かった） () 。


* 友人との対戦は、見知らぬ人との対戦に比べ、IBIが有意に短かった（＝心拍数が高かった） () 。





## 先行研究との差分

### 手法面

* 従来のプレゼンス研究が主にメディアの技術的・内容的特徴に焦点を当てていたのに対し、本研究は「共遊する相手（社会的要因）」を実験的に操作変数として組み込んだ点が異なります 。


* 自己報告だけでなく、心拍間隔（IBI）という生理指標を併用して覚醒レベルを客観的に評価しています 。



### 結果面

* ソーシャルサポートに関する先行研究では、友人の存在がストレス反応（生理的反応）を減衰させるとされていたが 、本研究のような競争的ゲームの文脈では、友人の存在が逆に覚醒（心拍数）を高めることが示されました 。


* この覚醒の増加は、友人がタスクパフォーマンスの継続的なリマインダーとして機能し、良いパフォーマンスをしたいという動機付けを高めるためであると解釈されています 。



## 議論

* 
**社会的競争と覚醒:** 他者との対戦は、高い「タスク遂行の評価の可能性（evaluation potential）」を伴うため、覚醒を高めると考えられます 。これは、対人対戦における脅威の予期が高かったことからも支持されます 。


* 
**友人と覚醒:** 友人との対戦で見知らぬ人よりも高い覚醒が見られたのは、友人がパフォーマンス比較の基準として機能し、負けたくない（良い成績を出したい）という重要性が増すためと考えられます 。


* 
**覚醒とプレゼンスの媒介関係:** 対人対戦（特に友人）において高い空間的プレゼンスが得られた要因として、覚醒が媒介している可能性があります 。覚醒は注意を高めるため 、ゲーム環境への注意資源の配分が増加し（現実環境への注意が減少し）、結果としてプレゼンスが高まったという解釈が、空間的プレゼンスの2レベルモデルと整合します 。


* 
**ポジティブな感情:** 人間相手のプレイで感情がポジティブになったのは、人間が社会的相互作用に対して本質的に報酬を感じる（接近動機を持つ）ためであると考えられます 。


* 
**限界と今後の課題:** 参加者が同じ部屋にいたため、インターネット経由など物理的に離れた状態でも同様の結果が得られるか（相手が誰か知っているだけで効果があるか）は今後の検証課題です 。また、性格（社交性など）の影響も検討の余地があります 。
