# Playing online games against computer- vs. human-controlled opponents: Effects on presence, flow, and enjoyment

* Earned empirical evidence that perceiving an opponent as human significantly enhances presence, flow, and enjoyment compared to perceiving an opponent as a computer, by conducting a deceptive experiment using an online game.
* オンラインゲームを用いた実験において、対戦相手が「コンピュータ」ではなく「人間」であると信じ込ませる欺瞞手法を用いることで、相手を人間だと認識することがプレゼンス（実在感）、フロー体験、および楽しさを有意に向上させるという実証的証拠を得た論文。

## 著者

* David Weibel 


* Bartholomäus Wissmath 


* Stephan Habegger 


* Yves Steiner 


* Rudolf Groner 



## 概要

本研究は、オンラインゲームにおいて対戦相手が「コンピュータ」であるか「他のユーザー（人間）」であるかの違いが、プレイヤーの心理的体験（プレゼンス、フロー、楽しさ）にどのような影響を与えるかを調査しました 。70名の大学生を対象に、実際にはどちらの条件でもコンピュータと対戦させるものの、実験群には「別室にいる人間と対戦している」と信じ込ませる実験を行いました 。その結果、対戦相手が人間であると認識した参加者は、コンピュータだと認識していた参加者に比べて、プレゼンス、フロー、楽しさのすべての指標で有意に高い数値を報告しました 。さらに分析により、フロー体験がプレゼンスと楽しさの関係を媒介していることが明らかになりました 。

## 背景

### 研究の意義

コンピュータゲームやビデオゲームの人気は急速に高まっており、特にインターネットを通じて他者と対戦できるオンラインゲームが普及しています 。これまで、対人対戦が楽しさを向上させることや、競争要素が重要であることは理論的に議論されてきましたが、実証的な結果は不足していました 。本研究は、対戦相手の属性（人間かコンピュータか）が、プレイヤーの没入感（プレゼンス）や集中状態（フロー）、そして楽しさに与える影響を実験的に検証することで、オンラインゲームの人気の要因である「社会的競争」の効果を明らかにすることに意義があります 。

### 関連研究

* 
**プレゼンス（Presence）:** 媒介された環境（バーチャル環境）に、物理的環境を超えて「そこにいる」と感じる主観的な感覚です 。対話性（Interactivity）が高いほどプレゼンスが高まるとされています 。


* 
**フロー（Flow）:** チクセントミハイによって提唱された概念で、活動そのものに完全に没頭し、自意識が消失し、時間感覚が歪むような本質的に満足度の高い精神状態を指します 。メディア利用やゲームプレイにおける楽しさの源泉とされています 。


* **対戦相手の影響:**
* Heeter (1992) は、バーチャル空間に他者が存在することで社会的プレゼンスや楽しさが向上すると仮定しました 。


* Mandrykら (2006) は、対人戦の方がコンピュータ戦よりも生理学的覚醒度や楽しさが高いことを示しました 。


* Ravajaら (2006) は、対人戦の方が空間的プレゼンスが高くなることを示しました 。


* 
**位置づけ:** これら先行研究の多くは対戦相手が同じ部屋にいる状況でしたが、本研究はオンラインゲームの特性（相手が見えない状況）に焦点を当てています 。





## 手法

### 概要

* 一要因多変量デザインを使用しました。要因は「対戦相手のタイプ（コンピュータ vs 人間）」です 。


* 実際には両群ともコンピュータと対戦しましたが、実験群には人間と対戦していると信じ込ませる欺瞞（Deception）を用いました 。



### 前提条件

* 
**参加者:** 心理学専攻の大学生83名（分析対象は操作チェック通過後の70名）。平均年齢23.9歳 。


* 
**使用ゲーム:** BioWare社の『Neverwinter Nights』（RPG）。MODツール（Aurora Toolset）により実験用にカスタマイズされました 。



### 詳細

* 
**グループ分け:** 性別をマッチングさせた上で、対照群（コンピュータと知らされる）と実験群（人間と知らされる）にランダムに割り当てました 。


* **教示の操作（欺瞞）:**
* 実験群の参加者には、「別室にいる他の参加者と対戦する」と説明しました 。


* 実験者が「相手が来るのを待つ」ふりをして退室し、しばらくして「相手が到着した」と告げる演出を行いました 。


* 対照群には、コンピュータと対戦すると通常通り説明しました 。




* **ゲームの手順:**
1. 
**探索パート（15分）:** バーチャル世界を旅し、「ルビー」を集めるタスク。戦闘は弱く設定された敵とのみ発生。操作に慣れるためのフェーズでもあります 。


2. 
**戦闘パート（5ラウンド）:** アリーナに移動し、対戦相手と決闘を行います 。




* **標準化と統制:**
* 生態学的妥当性は犠牲になりますが、統制のために戦闘結果は「すべてのプレイヤーが僅差で負ける」ように事前にプログラムされていました 。これにより、勝敗による感情への影響を排除しました。


* 対戦相手の名前表示について、対照群にはファンタジー風の名前、実験群には一般的な人間の名前が表示されるようにしました 。





### 評価方法

実験後、以下の尺度を用いた質問紙調査を行いました（5段階リッカート尺度）。

* 
**プレゼンス:** Kim & Biocca (1997) の尺度（8項目）。「到着（その世界にいる感覚）」と「出発（現実世界から離れる感覚）」の2次元を測定 。


* 
**フロー:** Rheinbergら (2003) の短縮版フロー尺度（10項目）。「スムーズな実行」と「没入（吸収）」の2次元を測定 。


* 
**楽しさ (Enjoyment):** 「ゲームを楽しめたか？」という単一項目で測定 。


* 
**操作チェック:** 実験後に「本当に人間（あるいはコンピュータ）と戦ったと信じていたか」を確認し、信じなかった参加者（対照群6名、実験群7名）を除外しました 。



## 結果と結論

### 概要

4つの仮説すべてが支持されました。対戦相手が人間であると信じてプレイすることは、実際に同じコンピュータプログラムと対戦しているにもかかわらず、プレゼンス、フロー、楽しさを有意に高めました。

### 詳細

* 
**仮説1（プレゼンス）の支持:** 人間と対戦していると思った参加者 () は、コンピュータと対戦した参加者 () よりも高いプレゼンスを報告しました ()。これが最も強い効果でした 。


* 
**仮説2（フロー）の支持:** 人間対戦条件 () は、コンピュータ対戦条件 () よりも高いフロー体験を報告しました () 。


* 
**仮説3（楽しさ）の支持:** 人間対戦条件 () は、コンピュータ対戦条件 () よりもゲームを楽しみました () 。


* 
**仮説4（相関と媒介）の支持:** プレゼンス、フロー、楽しさは互いに正の相関がありました 。


* 回帰分析の結果、フローがプレゼンスと楽しさの関係を「完全に媒介」していることが示唆されました 。つまり、高いプレゼンスが高いフローを引き起こし、その高いフローが楽しさにつながるというモデルです 。逆（プレゼンスがフローと楽しさを媒介する）は成立しませんでした 。





## 先行研究との差分

### 手法面

* 
**対戦相手の不在:** Mandrykら (2006) や Ravajaら (2006) の研究では対戦相手が同じ部屋にいましたが、本研究ではオンラインゲームの状況を模して、相手が見えない状況で行われました 。


* 
**完全な統制:** 相手が実際にはコンピュータであるため、相手のプレイスタイルや強さ、魅力といった交絡変数が排除され、純粋に「相手が人間であるという認識」の効果を測定できました 。



### 結果面

* プレイヤーが最終的に「負ける」ように設定されていたにもかかわらず、人間相手だと思っていた群の方が楽しさを感じていた点は興味深い発見です 。通常、負けは不満につながると考えられますが、社会的競争の魅力がそれを上回った可能性があります。



## 議論

* 
**社会的競争の影響:** オンラインゲームの人気の一因は、他者と競う「社会的競争」が没入感（プレゼンス）を高める点にあるという仮説が支持されました 。


* 
**eラーニングへの示唆:** 対話的なプレイがプレゼンスやフローを高めることから、eラーニングにおいても非対話的な形式より、他者との関わりがある対話的な形式が学習パフォーマンスや動機づけを高める可能性があります 。


* 
**フローの役割:** 楽しさはプレゼンスから直接生まれるのではなく、プレゼンスによって引き起こされた「フロー状態」の結果として生じることが示唆されました 。


* **限界:**
* 15分という制限時間は通常のゲームプレイとは異なる可能性があります 。


* 参加者が心理学の学生であり、一般的なゲーマー層とは属性が異なります 。


* 自己報告（質問紙）による測定であるため、社会的望ましさの影響を受ける可能性があります 。


* 統制された実験室環境であるため、実際の自発的なゲームプレイとは状況が異なります 。
