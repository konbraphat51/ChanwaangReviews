**Title:** Sample Size Justification
**Author:** Daniël Lakens

**内容の一文要約（日本語）:**
本論文は、量的実証研究におけるサンプルサイズ（標本サイズ）を正当化するための6つのアプローチを体系的に解説し、研究者が自身の推論目標に照らして収集データの「情報の価値」を最大化・評価するための包括的なガイドラインを提供するものである。

**One Sentence Summary (English):**
This overview article discusses six approaches to justify sample size in quantitative empirical studies and provides guidelines for evaluating the informational value of data based on inferential goals.

---

### 詳細要約

本論文は、研究計画における最も重要なステップの一つである「サンプルサイズの正当化」について、単に数字を述べるのではなく、なぜそのデータサイズが研究の推論目標に対して有益な情報をもたらすのかを論理的に説明するための枠組みを提供しています。

#### 1. 序論：情報の価値と推論目標

サンプルサイズの正当化における主要な目的は、収集されるデータが研究者の「推論目標（inferential goals）」に対して、いかに価値ある情報を提供するかを説明することにあります 。
研究者が助成金の申請や倫理審査においてサンプルサイズを記述する際、単に観察数を記載するだけで、その正当性が示されていないことが多々あります 。これは、その研究がどの程度有益（informative）であるかを評価することを困難にします 。データ収集後に有意差が出なかった場合などの問題を避けるため、データ収集前に慎重に正当化を行う必要があります 。

#### 2. サンプルサイズを正当化する6つのアプローチ

著者は、量的研究においてサンプルサイズを正当化するための6つのアプローチを提示しています 。

**① 全母集団の測定 (Measure Entire Population)**

* 
**適用:** 母集団が有限であり、その（ほぼ）すべてを測定可能な場合（例：特定の企業の全従業員、少数のトップアスリートなど） 。


* 
**論理:** 全データを利用したため、正当化は明白である 。全母集団を測定した場合、仮説検定は不要であり（一般化する対象がないため）、母集団効果量は既知となる 。


* 
**注意点:** 母集団の一部しか測定していない場合でも、母集団サイズ  が既知であれば、有限母集団修正（finite population correction）を行い、分散を縮小させることができる 。



**② リソースの制約 (Resource Constraints)**

* 
**適用:** 時間、資金、対象者の希少性などの現実的な制約が、収集可能なサンプルサイズを決定する主要因となる場合 。


* 
**論理:** 常にリソースは有限であり、科学者はコストとデータの情報の価値のトレードオフに直面する 。このアプローチでは、利用可能なリソースから収集可能な観察数  を算出し、その  で研究を行う価値があるか（有益か）を評価する 。


* **評価方法:**
* たとえ小規模なデータでも、コイン投げ（推測）よりは誤り率を減らせる場合、意思決定において価値がある 。


* 将来のメタ分析に寄与する可能性がある（ただし、データ共有と将来のメタ分析実施の蓋然性が必要） 。


* 
**推奨事項:** リソース制約による正当化を行う場合、最小の統計的に検出可能な効果量（minimal statistically detectable effect）や、信頼区間の幅、特定の研究領域で期待される効果量を考慮し、**感度パワー分析（sensitivity power analysis）**を行って、その  でどの程度の効果量を検出できるかを報告すべきである 。





**③ 事前パワー分析 (A-priori Power Analysis)**

* 
**適用:** 特定の効果量を、希望する統計的検出力（パワー）で棄却・検出することを目的とする場合 。


* 
**論理:** 指定した効果量、有意水準（）、検出力（）に基づき、必要な観察数を算出する 。


* 
**重要点:** 目的は「真の効果量」に対して十分なパワーを得ることではなく（真の値は不明なため）、「研究者が検出したいと仮定した効果量」に対して十分なパワーを得ることである 。


* 
**実施:** G*Powerなどのソフトウェアやシミュレーション（Rなど）を用いる 。複数の検定を行う場合（例：有意差検定と等価性検定の双方を行う場合）、すべての検定が有益となる最大のサンプルサイズを採用すべきである 。



**④ 精度（Accuracy / Planning for Precision）**

* 
**適用:** パラメータの推定精度に焦点を当て、信頼区間（CI）の幅を特定の範囲内に収めることを目的とする場合 。


* 
**論理:** 推定値の周囲の信頼区間が、希望する幅になるようにデータを収集する 。


* 
**課題:** どの程度の精度（CIの幅）を目指すべきかについての明確な指針や文献が不足している 。研究者は、ある範囲以下の測定差が推論に影響を与えないという規範（norm）に基づいて精度を決定する必要があるかもしれない 。



**⑤ ヒューリスティクス (Heuristics)**

* 
**適用:** 文献や一般的な規範（ルールオブサム）に基づく場合 。


* 
**評価:** 多くの場合、推奨されない。「1セルあたり20人」「全体で50人」といったルールは、しばしば誤った引用や単純化に基づいている 。


* **例:** Simmons et al. (2011)は「少なくとも20人」と言ったが、後に「効果が大きくない限り 」と修正している。しかし、前者の数字だけが一人歩きしている場合がある 。


* 
**例外:** 特定の研究領域で、意味のある効果量の閾値について合意が形成されている場合などは、有効なヒューリスティクスになり得る 。



**⑥ 正当化なし (No Justification)**

* 
**適用:** 明確な推論目標がなく、特定のサイズを選んだ理由がない場合 。


* 
**論理:** 偽の正当化をするよりも、正直に「正当化はない」と述べる方が誠実である 。ただし、その場合でも、収集されたデータで検出可能だった効果量（感度分析）や信頼区間の幅について事後的に評価し、報告することが推奨される 。



#### 3. 「興味深い効果量」をどう評価するか

サンプルサイズを決定する際、研究者は「どの程度の効果量が興味深いのか」を評価する必要があります。著者は以下の視点を提示しています 。

1. **最小の関心ある効果量 (Smallest Effect Size of Interest: SESOI):**
* 理論的予測や実用的な重要性に基づき、これより小さいと意味がないとみなされる最小の効果量 。これを事前に設定できれば、SESOIを検出（または棄却）するための十分なパワーを持つ設計が可能となり、最も強力な正当化となる 。




2. **最小の統計的に検出可能な効果 (Minimal Statistically Detectable Effect):**
* 与えられたサンプルサイズと有意水準において、統計的に有意となり得る最小の効果量（クリティカル効果量） 。


* リソース制約などで  が決まっている場合、この値を確認し、それが現実的に期待できる効果量の範囲内かを自問すべきである 。




3. **期待される効果量 (Expected Effect Size):**
* 理論、メタ分析、先行研究に基づく予測値 。


* 
**メタ分析の利用:** 出版バイアスや異質性（heterogeneity）に注意が必要。自身の研究に一般化可能か慎重に検討すべき 。


* 
**先行研究の利用:** パイロット研究などの小規模な先行研究の効果量をそのままパワー分析に使うことは危険である。出版バイアスやランダムな変動により、効果量が過大評価されている可能性が高いためである 。出版された有意な結果のみに基づく効果量推定は、切断された分布（truncated distribution）からの推定となり、バイアスがかかっている 。




4. **信頼区間の幅 (Width of Confidence Interval):**
* 効果がない（）と仮定した場合、どの範囲の効果量までなら棄却できるかを確認する 。




5. **感度パワー分析 (Sensitivity Power Analysis):**
* 
 が固定されている場合、横軸に効果量、縦軸にパワーをとった曲線をプロットし、どの範囲の効果量に対して十分な検出力があるかを視覚的に評価する 。




6. **研究領域における効果量の分布 (Distribution of Effect Sizes in a Research Area):**
* 特定の分野で経験的に観察される効果量の範囲。ただし、単に「心理学の平均は 」といった平均値を使うことは推奨されない（異質性が高いため） 。





#### 4. その他、有益な研究を設計するための考察

* **妥協パワー分析 (Compromise Power Analysis):**
* リソースの制限で  が増やせない場合、慣習的な  / Power=0.80 に固執するのではなく、Type I エラー（偽陽性）と Type II エラー（偽陰性）のコストを比較検討し、エラー率のバランスを調整する 。


* 例えば、Type I エラーと Type II エラーの重大さが等しいと判断する場合、 を上げて検出力を高める（エラー率を均衡させる）ことが合理的な場合がある 。




* **事後パワー分析（Post-hoc Power）の禁止:**
* 収集されたデータの観察された効果量に基づいて計算される「事後パワー」は、p値を別の形で表現しているに過ぎず（p=0.05なら事後パワーは50%になる）、無意味であるため報告すべきではない 。代わりに感度パワー分析を行うべきである 。




* **逐次分析 (Sequential Analyses):**
* データ収集の中間段階で分析を行い、結果に応じてデータ収集を継続・停止する手法。効率的であり、サンプルサイズを節約できる可能性がある 。




* **サンプルサイズを増やさずにパワーを上げる方法:**
* 方向性のある仮説がある場合の片側検定の採用（妥当性には議論があるが、ネイマン・ピアソンの観点からは正当化可能） 。


* 参加者間デザインではなく、参加者内デザイン（within-participant design）を採用する。相関が高いほどパワーが上がる 。


* 測定の信頼性を高める、共変量を利用する 。





#### 5. 質的研究におけるサンプルサイズ

質的研究においても「情報の価値」の視点は適用可能である。一般的には「飽和（saturation）」（新しい情報が得られなくなる時点）が基準となる 。母集団内のコード（概念）の数や出現確率に基づき、必要なサンプルサイズを推定する試みも提案されている 。

#### 6. 結論

サンプルサイズの正当化は、単なる事務的な手続きではなく、研究者が「この研究から何を学びたいのか」を明確にし、そのために必要な情報を収集できるかを評価するプロセスである 。十分な情報価値がないと判断される場合は、データ収集を行わない（リソースの浪費を防ぐ）という判断も重要である 。

### 質疑応答

**Q: この論文には、具体的な推奨サンプルサイズ（例：心理学なら100人が良いなど）は書かれていますか？**
**A:** 書かれていません。著者は、一般的なルール（ヒューリスティクス）や「中程度の効果」といったデフォルト値に依存することを批判しており、個々の研究の推論目標、リソース、期待される効果量に基づいて、個別に正当化を行うべきだと主張しています。

**Q: パイロット研究の結果を使って本実験のサンプルサイズを決めることは推奨されていますか？**


**A:** 推奨されていません。著者は、小規模なパイロット研究から得られる効果量推定値は不正確であり、出版バイアスや「有意な結果のみをフォローアップする」というバイアスにより、効果量が過大評価される傾向がある（切断された分布になる）と警告しています 。これにより、本実験のパワーが不足するリスクがあります。

**Q: サンプルサイズ計算に使えるツールについて言及はありますか？**


**A:** はい、言及されています。具体的には、G*Power 、Rのパッケージ（`pwr`, `superb` , `MBESS` , `BUCSS` , `rpact` ）、および著者が作成したShinyアプリ  などが紹介されています。
